   - Guard: if `value == null` return `'0 %'`
   - Round to 1 decimal: `Math.round(value * 10) / 10`

   - If has decimal: `rounded.toLocaleString('de-DE') + ' %'`

   - Default to `'Unbekannt'` if falsy

   - Replace remaining non-alphanumeric chars (except hyphens) with `-`
   - Collapse multiple hyphens, trim leading/trailing hyphens
   - Truncate to 50 chars

**`generateForSingleCreditor(client, snapshot, creditor, outputDir)`**:
1. Determine `planType = snapshot.plan_type || 'NULLPLAN'`
2. Select template: `this.templatePaths[planType] || this.templatePaths.NULLPLAN`
3. Read template: `const templateContent = await fs.readFile(templatePath)`
4. Create PizZip: `const zip = new PizZip(templateContent)`
5. Normalize quotes: `this._normalizeGermanQuotes(zip)`
6. Create Docxtemplater with **exact same config** as first round: `{ paragraphLoop: true, linebreaks: true, delimiters: { start: '{', end: '}' } }`
7. Prepare data: `const templateData = this.prepareTemplateData(client, snapshot, creditor)`
8. Render: `doc.render(templateData)`
9. Generate buffer: `doc.getZip().generate({ type: 'nodebuffer', compression: 'DEFLATE' })`
10. Build filename: `this._buildFilename(creditor, client, planType)`
11. Write file: `await fs.writeFile(outputPath, outputBuffer)`
12. Return `{ success: true, creditor_id, creditor_name, filename, path: outputPath }`

4. For each creditor: try `generateForSingleCreditor`, push to results; catch push to errors (continue-on-failure per CONTEXT.md locked decision)
