  <action>
Create the RTK Query API layer following the pattern from the old frontend (`src/store/`) but simplified for the new admin-only frontend.

1. **src/store/api/baseApi.ts** — Create the base API:
   ```typescript
   import { createApi, fetchBaseQuery } from '@reduxjs/toolkit/query/react';

   const baseQuery = fetchBaseQuery({
     baseUrl: import.meta.env.VITE_API_BASE_URL || '',
     prepareHeaders: (headers) => {
       const token = localStorage.getItem('admin_token');
       if (token) {
         headers.set('Authorization', `Bearer ${token}`);
       }
       return headers;
     },
   });
   ```
   - Use `import.meta.env.VITE_API_BASE_URL` (Vite env var pattern) instead of `process.env` (CRA pattern)
   - When VITE_API_BASE_URL is empty (dev), requests go to `/api/...` which Vite proxies to localhost:3001
   - When VITE_API_BASE_URL is set (prod), requests go to `https://prod-url.com/api/...`
   - Admin-only: Only check `admin_token` in localStorage (not auth_token/portal_session_token — this is an admin-only app)
   - Create the API with `createApi`:
     - `reducerPath`: `'api'`
     - `baseQuery`: the baseQuery above (no reauth wrapper yet — that's Phase 20 auth work)
     - `tagTypes`: `['Client', 'Document', 'Creditor', 'WorkflowStatus']`
     - `endpoints`: empty `() => ({})` — endpoints injected in Phase 21+
   - Export `baseApi`

2. **src/store/index.ts** — Create the Redux store:
   ```typescript
   import { configureStore } from '@reduxjs/toolkit';
   import { baseApi } from './api/baseApi';

   export const store = configureStore({
     reducer: {
       [baseApi.reducerPath]: baseApi.reducer,
     },
     middleware: (getDefaultMiddleware) =>
       getDefaultMiddleware().concat(baseApi.middleware),
   });

   export type RootState = ReturnType<typeof store.getState>;
   export type AppDispatch = typeof store.dispatch;
   ```

3. **src/store/hooks.ts** — Create typed hooks:
   ```typescript
   import { useDispatch, useSelector } from 'react-redux';
   import type { RootState, AppDispatch } from './index';

   export const useAppDispatch = useDispatch.withTypes<AppDispatch>();
   export const useAppSelector = useSelector.withTypes<RootState>();
   ```

4. **src/main.tsx** — Add Redux Provider:
   - Import `{ Provider }` from `react-redux`
   - Import `{ store }` from `./store`
   - Wrap the app: `<Provider store={store}><BrowserRouter><App /></BrowserRouter></Provider>`
   - **IMPORTANT:** If Plan 19-02 has already run and added BrowserRouter, keep that. If not, just add Provider around whatever is there. The key constraint is Provider must wrap BrowserRouter which wraps App. Since both plans depend on 19-01 and this file may be edited by both, read the current state first.
