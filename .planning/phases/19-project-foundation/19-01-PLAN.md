1. **package.json** — Make these changes:
   - Change `"name"` from `"@figma/my-make-file"` to `"rasolv-admin"` per user decision
   - Move `react` (18.3.1) and `react-dom` (18.3.1) from `peerDependencies` to `dependencies` (remove from peerDependencies and peerDependenciesMeta entirely)
   - Add `@reduxjs/toolkit` (^2.2.0) and `react-redux` (^9.1.0) to dependencies — needed for RTK Query in Plan 19-03
   - Add `"preview": "vite preview"` to scripts
   - Add `"typecheck": "tsc --noEmit"` to scripts

2. **tsconfig.json** — Create with these options (relaxed for Figma-generated code):
   - `target`: "ES2020"
   - `module`: "ESNext"
   - `moduleResolution`: "bundler"
   - `jsx`: "react-jsx"
   - `strict`: false (Figma-generated code has implicit anys and loose types)
   - `noEmit`: true (Vite handles transpilation)
   - `skipLibCheck`: true
   - `esModuleInterop`: true
   - `allowImportingTsExtensions`: true
   - `isolatedModules`: true
   - `paths`: `{ "@/*": ["./src/*"] }` to match existing vite.config.ts alias
   - `include`: ["src"]
   - `references`: [{ "path": "./tsconfig.node.json" }]

3. **tsconfig.node.json** — Create for Vite config files:
   - `compilerOptions`: target ES2022, module ESNext, moduleResolution bundler, allowSyntheticDefaultImports true, noEmit true
   - `include`: ["vite.config.ts"]

4. **.env.example** — Create with:
   ```
   # API Base URL — used by RTK Query fetchBaseQuery
   # Development: http://localhost:3001 (or use Vite proxy, so leave empty)
   # Production: https://mandanten-portal-docker.onrender.com
   VITE_API_BASE_URL=
   ```

5. **.env.development** — Create with:
   ```
   # Dev uses Vite proxy — API calls go to /api/* which Vite proxies to localhost:3001
   VITE_API_BASE_URL=
   ```

6. **index.html** — Update `<title>` from "Untitled" to "Rasolv Admin"

After editing, run `cd MandantenPortalDesign && npm install` to install react, react-dom as real dependencies plus the new Redux packages.
