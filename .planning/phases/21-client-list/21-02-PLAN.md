**Replace the local useState-based filter state from Plan 21-01 with URL search params.**

1. **URL param sync using `useSearchParams` from react-router:**
   - Import `useSearchParams` from `react-router` (or `react-router-dom` depending on which is installed)
   - Read all filter state from URL:
     ```typescript
     const [searchParams, setSearchParams] = useSearchParams();
     const search = searchParams.get('search') || '';
     const statusFilter = searchParams.get('status') || 'all';
     const flowFilter = searchParams.get('flow') || 'all';
     const page = parseInt(searchParams.get('page') || '1', 10);
     const limit = parseInt(searchParams.get('limit') || '25', 10);
     ```
   - Create a helper to update URL params without losing other params:
     ```typescript
     const setParam = (key: string, value: string) => {
       setSearchParams(prev => {
         const next = new URLSearchParams(prev);
         if (value === '' || value === 'all' || (key === 'page' && value === '1') || (key === 'limit' && value === '25')) {
           next.delete(key); // Clean URL: omit default values
         } else {
           next.set(key, value);
         }
         // Reset page to 1 when changing filters/search (not when changing page itself)
         if (key !== 'page' && key !== 'limit') {
           next.delete('page');
         }
         return next;
       });
     };
     ```
   - Remove the local `useState` calls for page, limit, search, statusFilter, flowFilter from Plan 21-01. Replace with URL-derived values.

2. **Debounced search:**
   - Keep a local `searchInput` state for the controlled input (immediate typing feedback)
   - Sync to URL param `search` with ~300ms debounce:
     ```typescript
     const [searchInput, setSearchInput] = useState(search); // Initialize from URL
     useEffect(() => {
       const timer = setTimeout(() => {
         setParam('search', searchInput);
       }, 300);
       return () => clearTimeout(timer);
     }, [searchInput]);
     ```
   - When URL `search` param changes externally (e.g., back button), sync back to `searchInput`:
     ```typescript
     useEffect(() => {
       setSearchInput(search);
     }, [search]);
     ```
