4. **Add handleNewRowFieldChange** function to track field values for unsaved rows:
   ```typescript
   const handleNewRowFieldChange = (tempId: string, fieldName: string, value: string) => {
     setNewRows(prev => prev.map(r =>
       r.tempId === tempId ? { ...r, fields: { ...r.fields, [fieldName]: value } } : r
     ));
   };
   ```

Remove the row from `newRows` after a brief green flash (use setTimeout 1000ms).
   - On error: Set `saving: false`, `error: 'Speichern fehlgeschlagen'` on the row.

     - `value={newRow.fields[fieldName] || ''}`
